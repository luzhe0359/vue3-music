## 运行步骤
1. 克隆项目
```bash
# git
git clone https://github.com/luzhe0359/vue3-music.git
```

2. 安装依赖

```bash
# pnpm (推荐)
pnpm install

# npm
npm install

# yarn
yarn install
```

3. 运行项目

```bash
# pnpm (推荐)
pnpm run dev

# npm
npm run dev

# yarn
yarn dev
```

4. 查看模板

运行完毕，自动打开 http://127.0.0.1:8088/

5. 其他

其他配置，请自行修改/添加
